Installation / Start

Zur Installation den MIF-Editor aus dem Internet herunterladen (ggf. entpacken) und in ein beliebiges Verzeichnis speichern.
Zum Starten der Software die Datei "MIF-Editor.exe" öffnen. (erkennbar an dem Raketensymbol)

Erste Schritte

Nach dem Starten sehen Sie rechts "Bild 1" bis "Bild 24". Klicken Sie verschiedene Bilder an. Die gelbe Markierung gibt an, welches Bild Sie gerade markiert haben.
Rechts oben sehen Sie die Eigenschaften des Bildes. Die Schwarze Schrift können Sie genauso anklicken wie die Bilder. Auch hier wird 3in gelber Rahmen angezeigt. Klicken Sie auf "Bildbreite" und geben Sie dort "250" ein. (Sie können das Numpad dabei nicht benutzen. Verfahren Sie mit dem Feld "Bildhöhe" genauso.

Sie sehen jetzt unten rechts im Vorschaufenster ein 250 x 250 Pixel großes, schwarzes Rechteck. Klicken Sie zwei mal auf "Farbe 1". Jetzt öffnet sich die Farbauswahlbox und Sie können ihre Lieblingsfarbe auswählen und auf "OK" klicken. Das Rechteck hat jetzt diese Farbe übernommen.

Klicken Sie zewi mal auf "Speicherart: Farbe" und wählen Sie "Farbverlauf (Horizontal)" aus der Liste. Sie sehen jetzt einen Farbverlauf von Ihrer Lieblingsfarbe nach schwarz. Unter "Farbe 2" können Sie die Farbe einstellen, zu welcher der Farbverlauf gehen soll. Nehmen Sie eine weitere Farbe, die Ihnen gut gefällt.

Ihnen wird aufgefallen sein, dass bei dem Bild, dass Sie rechts angeklickt haben, steht: "Farbverlauf (Horizontal)". Dies ist die Speicherart. Es werden nur Bilder mit Inhalt (Also größer al 0 x 0 Pixel) gespeichert. Bei allen Bildern, wo das der Fall ist, wird die Speicherart angezeigt. Drücken Sie nun auf [S] und Speichern Sie das Bild ab.
Sie haben gerade Ihr erstes MIF-Bild gespeichert. Herzlichen Glückwunsch.

Farben / Farbverläufe

Sie können im MIF-Format Farben bzw. Farbverläufe als Farbe bzw. Farbverlauf abspeichern. Das bedeutet, das ein Bild mit einer Farbe oder einem Farbverlauf maximal 13 Byte groß ist egal wie viele Pixal das Bild hat. Stellen sie dazu "Bildbreite" und "Bildhöhe" auf die gewünschte Größe ihres Bildes ein. Die Anzahl an Frames sollten Sie dabei auf dem Standartwert "1" belassen. Unter "Speicherart" stellen Sie den gewünschten Modus ein und mit "Farbe 1" und "Farbe 2" bestimmen sie die Farbe ihres Bildes.

Bilder

Stellen Sie zunächst die "Speicherart" auf ein beliebiges Bild und klicken Sie zwei mal auf "Dateiname"
Danach laden Sie ein beliebiges Bild von Ihrer Festplatte. Dieses wird danach im Vorschaufenster angezeigt. Als nächstes stellen Sie die Speicherart ein. Dabei gibt es drei möglichkeiten ein Bild einzufügen:
  1. Als unkomprimiertes Bild: Ähnlich wie bei Bitmaps. Das Bild verliert keine Informationen aber der Speicherbedarf kann groß werden.
  2. Als LWZ-Komprimiertes Bild: Diese Kompression fasst gleiche, hintereinander liegende Pixel zusammen. Das Bild wird dabei verlustfrei komprimiert. Diese kompression funktioniert allerdings nicht bei eingescannten Bildern oder Fotos und braucht dann soger noch mehr Speicherplatz wie ein unkomprimiertes Bild.
  3. Als YUV-Komprimiertes Bild: Hierbei werden Farbinformationen entfernt, weil das Menschliche Auge diese nicht so gut wahrnehmen kann. Diese kompression reduziert die Datenmenge immer um die Hälfte, jedoch wird die Qualität des Bildes geringfügig schlechter.
Die richtige Kompression findet man am Besten durch ausprobieren.

Erweiterte Schritte

Eine Hintergrundfarbe stellen Sie unter "Transparente Farbe" ein. Diese wird dann ausgeschnitten.
Beachten Sie: Manche Programme unterstützen unter Umständen keine transparente Farbe. In diesem Fall bleibt die Einstellung wirkungslos.

Unter "Frames" stellen Sie ein, dass Ihr Bild in mehrere Teile zerlegt wird. Bei manchen Spielen macht man so Animationen.
Achtung: Dieses Feature ist z.Z. noch nicht ganz ausgereift, sodass Bilder mit mehreren Frames nicht lesbar sein können.

Grenzen dieser Version

  • Mit dieser Version lassen sich maximal 24 Bilder in einer Datei unterbringen, obwohl das MIF-Format bis zu 65.535 erlaubt.
  • Die Framerate lässt sich nicht einstellen und ist auf "1" festgelegt.
  • Maximale Bildbreite sowie Bildhöhe sind gemäß MIF-Format 65.535 Pixel. Die Frameanzahl ist nach dem MIF-Format auf 255 beschränkt.

Bugs

Die Zerlegung in mehrere Frame funktioniert nicht richtig.
