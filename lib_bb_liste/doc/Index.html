2.) Was ist "Liste"?

"Liste" ist eine Funktionsbibilothek für Blitz. Blitz ist eine Programmiersprache, die von der Syntax her Basic ähnelt und mit der sich relativ einfach Spiele programmieren lassen. Blitz-Website: www.blitzbasic.com

3.) Wer hat's verbrochen?

Die Standartversion Funktionsbibilothek wurde von Diego Semmler (www.dsemmler.de) geschrieben.

4.) Lizenz

Liste ist Open-Source, d.h. jeder darf den Code frei verwenden, anpassen oder weiter entwickeln. Wenn Sie eine Weiterentwicklung oder Modifikation haben, wüde ich mich sehr freuen, wenn Sie eine E-Mail an schicken würden.
